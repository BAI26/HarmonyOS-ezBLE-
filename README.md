# HarmonyOS-ezBLE-
⚡️ 专为 HarmonyOS Next 打造的原生全能物联网调试终端。集 Classic BT、BLE、USB 串口与 MQTT 于一身，支持实时波形、IMU 3D 姿态同步及虚拟仪表盘。A native, all-in-one IoT debugging terminal for HarmonyOS Next. Features Quad-mode connectivity (BT/BLE/USB/MQTT), real-time oscilloscopes, IMU 3D visualization, and virtual dashboards.
